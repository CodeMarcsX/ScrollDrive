# ScrollDrive üèéÔ∏è

**ScrollDrive** is a modern front-end animation project built with **HTML, CSS, and JavaScript**, focused on **high-performance scroll-based motion** and **premium visual design**.

The project showcases **cards moving in an arc layout**, synchronized with scroll, inspired by **luxury automotive motion** and **clean, high-end UI principles**.

---

## ‚ú® Features

* **Scroll-Based Card Motion**  
  Cards move along a smooth arc path, fully synchronized with scroll using GSAP.

* **Sticky Scroll Section**  
  Pin-based section with controlled scroll duration for cinematic motion.

* **Animated Counter**  
  Dynamic counter that updates based on the active card using Intersection Observer.

* **Split Text Animation**  
  Line-based text reveal animation triggered by scroll for a premium finishing effect.

* **Smooth Scrolling**  
  Natural and fluid scroll behavior powered by Lenis.

* **Responsive Motion Logic**  
  Motion radius and layout adapt dynamically to screen size.
